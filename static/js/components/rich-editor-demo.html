富文本编辑器组件示例

可复用的富文本编辑器组件，基于 Quill.js 构建

基本使用示例

最简单的富文本编辑器使用方式：

// 基本使用 const editor = new RichEditor({ container: '#editor1', placeholder: '请输入内容...', height: 200 });

高级配置示例

自定义工具栏和事件处理：

// 高级配置 const editor = new RichEditor({ container: '#editor2', placeholder: '请输入内容...', height: 300, toolbar: [ ['bold', 'italic', 'underline'], ['link', 'image'], [{ 'list': 'ordered'}, { 'list': 'bullet' }], ['blockquote', 'code-block'], [{ 'header': [1, 2, 3, false] }], [{ 'color': [] }, { 'background': [] }], [{ 'align': [] }], ['clean'] ], onChange: function(content) { console.log('内容变化:', content); }, onReady: function(editor) { console.log('编辑器就绪'); } });

只读模式示例

只读模式的富文本编辑器：

// 只读模式 const editor = new RichEditor({ container: '#editor3', placeholder: '只读内容...', height: 200, readonly: true, value: '<p>这是只读内容，用户无法编辑。</p>' });

功能特性

  • 基于 Quill.js 构建，功能强大
  • 支持多种文本格式（粗体、斜体、下划线等）
  • 支持链接和图片插入
  • 支持列表（有序、无序）
  • 支持代码块和引用
  • 支持标题格式
  • 支持文字颜色和背景色
  • 支持文本对齐
  • 支持只读模式
  • 支持内容变化监听
  • 支持多种内容格式获取（HTML、纯文本、Delta）
  • 支持程序化内容操作
  • 响应式设计，支持移动端
  • 支持暗色主题

API 文档

构造函数参数

new RichEditor({ container: '#editor', // 容器选择器 placeholder: '请输入内容...', // 占位符文本 height: 300, // 编辑器高度 toolbar: [...], // 工具栏配置 onChange: function(content) {}, // 内容变化回调 onReady: function(editor) {}, // 编辑器就绪回调 value: '', // 初始内容 readonly: false // 是否只读 });

主要方法

// 获取内容 editor.getContent('html'); // 获取HTML格式 editor.getContent('text'); // 获取纯文本 editor.getContent('delta'); // 获取Delta格式 // 设置内容 editor.setContent(content, 'html'); // 清空内容 editor.clear(); // 插入文本 editor.insertText('文本', { bold: true }); // 插入HTML editor.insertHTML('<p>HTML内容</p>'); // 格式化选中文本 editor.format('bold', true); // 获取选中文本格式 editor.getFormat(); // 设置只读模式 editor.setReadOnly(true); // 聚焦/失焦 editor.focus(); editor.blur(); // 获取编辑器实例 editor.getEditor(); // 销毁编辑器 editor.destroy();
